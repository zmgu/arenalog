# Changelog

## v1.0.0 (2026-05-31)

첫 번째 릴리즈.

### 주요 기능

- **챔피언 트래커** — 60챔피언 우승 진척도 표시 (6열 그리드, 골드/보라/회색 테두리 구분)
- **매치 히스토리** — 아레나 큐(1700/1750) 자동 수집, 증강·아이템 상세 펼침, 툴팁
- **편집 모드** — 수동 추가/삭제, 변경 내역 확인 팝업, 초보자용 단계별 가이드
- **검색** — 한글/영어/초성/한→영 오타 변환 실시간 필터링
- **새로고침** — 버튼 클릭 시 매치 히스토리 재조회, 신규 1등 챔피언 알림 팝업
- **다중 계정** — 계정 변경 시 puuid별 독립 데이터 관리
- **오프라인 캐시** — 앱 실행 시 챔피언·아이템·증강 아이콘 전체 메모리 캐시
